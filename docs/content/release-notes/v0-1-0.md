First release of `crossref`, a command-line tool for querying
[Crossref](https://www.crossref.org/) scholarly metadata by DOI.

## What is Crossref?

Crossref is the world's largest DOI registration agency for scholarly works.
Its REST API provides open metadata for journal articles, books, datasets,
conference papers, and more. No API key is required; this tool opts into the
polite pool automatically by including a `mailto:` address in the User-Agent.

## Commands

- **`crossref search <query>`** — search works by full text, with optional
  `--type` filter (e.g. `--type journal-article`).
- **`crossref work <doi>`** — fetch a single work by DOI. Accepts raw DOIs
  or `https://doi.org/` prefixed forms.
- **`crossref journal <issn>`** — fetch journal metadata by ISSN.
- **`crossref journals <query>`** — search journals by title keyword.
- **`crossref types`** — list all Crossref work type identifiers.

## Output formats

All commands support `-o table|json|jsonl|csv|tsv|url|raw`, `--fields`,
`--template`, and `-n` for result count. On a terminal, `table` is the default;
when piped, `jsonl` is the default.

## Example

```sh
crossref search "CRISPR gene editing" -n 5 -o table
crossref work 10.1038/nature12373
crossref journal 0028-0836
crossref journals "machine learning" -n 10
crossref types
```

## Authors

```
crossref search "Einstein relativity" -o jsonl | jq .authors
```
